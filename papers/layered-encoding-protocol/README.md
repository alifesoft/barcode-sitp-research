# Layered Encoding Protocol Paper

This directory contains materials related to the paper:

**Yurii Kulakov, Oleksandr Havryliuk**
**A Review of Data Encoding Problems in Barcodes and a Layered Encoding Protocol Based on the OSI Model**

## Overview

This paper analyzes structural limitations of data encoding in barcode systems and proposes a layered interpretation of barcode payload processing based on the OSI reference model.

The main focus of the paper is the Self-Identified Transport Protocol (SITP), a lightweight transport-layer mechanism for explicit identification of nested barcode payload encoding schemes. The proposed approach is intended to improve protocol identification, support extensible payload formats, and preserve compatibility with existing 2D barcode symbologies by operating at the application level.

## Directory purpose

This directory is intended to store paper-specific materials, including:

* selected experimental results used in the manuscript;
* tables and figures prepared for publication;
* supplementary notes related to the paper;
* revision materials and reviewer-response artifacts, if applicable;
* processed outputs generated by experimental software from other parts of the repository.

Raw experimental generation code and general SITP stability tools are maintained outside this directory, primarily in the `sitp-stability` directory.

## Relation to the repository

This paper is part of the broader `barcode-sitp-research` repository, which contains research code and experimental materials for SITP-based barcode encoding, ESID design, and related academic studies.

This directory is paper-oriented. It should contain only materials directly related to the layered encoding protocol article, while shared code, reusable datasets, and general experimental tools should remain in their corresponding repository-level directories.

## Status

The materials in this directory are research-oriented and may change as the manuscript is revised, extended, or prepared for publication.
