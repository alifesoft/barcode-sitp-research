# SITP Stability Analysis Paper

This directory contains materials related to a planned academic paper on the stability analysis of the Self-Identified Transport Protocol (SITP).

## Overview

The paper is currently in preparation. Its expected focus is the analysis of SITP behavior under different payload configurations, protocol parameters, checksum modes, and identification conditions.

## Directory purpose

This directory is intended to store paper-specific materials, including selected experimental results, tables, figures, notes, and processed outputs used during manuscript preparation.

General experimental code and raw generated data are maintained outside this directory, primarily in the `sitp-stability` directory.
